# Notifications List

## Admin Notifications

1. **Contact Form Submission** — New contact inquiry from user
2. **Dispute Raised** — New dispute raised on an order
3. **KYC Submission** — Freelancer submitted KYC documents
4. **New User Registered** — New freelancer or creator signed up
5. **Order Created** — New order created (payment confirmed)
6. **Withdrawal Request** — Freelancer submitted a withdrawal request

---

## Auth Notifications (User)

7. **Email Verification OTP** — OTP sent during registration
8. **Password Reset OTP** — OTP sent for password reset

---

## Creator Notifications

9. **Account Suspended** — Creator account suspended by admin
10. **Account Restored** — Creator account suspension lifted
11. **Deadline Extension Requested** — Freelancer requested more time
12. **Delivery Received** — Freelancer submitted a delivery
13. **Dispute Resolved** — Admin resolved a dispute
14. **Hire Request Sent (Confirmation)** — Hire request sent to freelancer
15. **Hire Request Accepted** — Freelancer accepted hire request
16. **Hire Request Declined** — Freelancer declined hire request
17. **Offer Received** — Freelancer sent a custom offer
18. **Invoice Email** — Invoice attached after order approved
19. **Order Completed** — Order marked complete after approval
20. **Payment Confirmed** — Payment received and held in escrow
21. **Dispute Raised (Confirmation)** — Creator raised a dispute
22. **Rating Request** — Prompt to rate freelancer after order

23. **Welcome Email** — Account created successfully

---

## Freelancer Notifications

24. **Account Suspended** — Freelancer account suspended by admin
25. **Account Restored** — Freelancer account suspension lifted
26. **Deadline Extension Accepted** — Creator approved extension request
27. **Deadline Extension Rejected** — Creator declined extension request
28. **Delivery Submitted (Confirmation)** — Delivery submitted successfully
29. **Dispute Raised Against You** — Creator raised a dispute against freelancer
30. **Dispute Resolved** — Admin resolved a dispute
31. **Hire Request Received** — Creator sent a hire request
32. **KYC Approved / Rejected** — Admin reviewed KYC documents
33. **Offer Accepted** — Creator accepted freelancer's custom offer
34. **Offer Rejected** — Creator declined freelancer's custom offer
35. **Offer Sent (Confirmation)** — Custom offer sent to creator
36. **Order Activated** — Payment confirmed, order is now active
37. **Order Approved** — Creator approved delivery
38. **Payment Released** — Earnings released to wallet
39. **Rating Request** — Prompt to rate creator after order
40. **Welcome Email** — Account created successfully
41. **Withdrawal Approved** — Withdrawal request approved by admin
42. **Withdrawal Request (to Admin)** — Admin notified of new withdrawal request
